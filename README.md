# 百托

![](https://github.com/ChangWenWu/baituo/blob/master/Delivery-Second/ic_launcher-web.png?raw=true)
    
**百托**APP实现了校园外卖订单的多级配送,包括3个客户端和一个服务器端:<br> 
* 商家客户端，实现外卖订单的创建和发送 <br> 
* 配送员客户端，实现外卖订单的状态切换 <br> 
* 用户客户端，实现订单查询，抢单<br>
